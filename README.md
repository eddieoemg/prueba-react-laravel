# Prueba React + Laravel (Autentificacion Oauth2)

# Instalación Laravel

- composer install

- php artisan migrate:fresh
- php artisan cache:table
- php artisan migrate

- php artisan migrate --path=database/migrations/stores
- php artisan migrate --path=database/migrations/people

- php artisan db:seed

- php artisan passport:client --personal
- php artisan serve


# Instalación React
- Ajustar la ruta del  archivo (.env)

- npm install
- npm run dev
